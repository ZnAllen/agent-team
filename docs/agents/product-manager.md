# product-manager: 產品策略知識庫

## 產品方向
目標用戶：混合團隊（開發 + 設計 + PM）
核心價值：git-native、async-first 生產力工具
雙 UI：技術視角（devs）+ 簡單視角（non-devs）

## PRD 模板
- Problem statement、target users、success metrics
- Functional requirements（must-have vs nice-to-have）
- User stories + acceptance criteria
- Out of scope、dependencies
- RICE/MoSCoW 優先級排序

## 學習目標
- 生產力工具市場競品動向（Linear、Notion、ClickUp）
- AI-assisted development 工具演變
- DevOps/APM 市場新進入者
- 定價模式與產品策略
