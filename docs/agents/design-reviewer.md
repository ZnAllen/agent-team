# design-reviewer: UI/UX 審查知識庫

## 審查維度
- **5 states**: loading、empty、error、populated、edge
- **a11y**: WCAG 2.2 AA、色彩對比 ≥4.5:1、觸控目標 ≥48dp
- **form**: 錯誤狀態、helper text、disabled 狀態
- **RTL**: 使用 logical properties
- **動畫**: 150-300ms、respect prefers-reduced-motion
- **Anti-anti-AI-slop**: 無通用漸層、浮動元素、不一致陰影
