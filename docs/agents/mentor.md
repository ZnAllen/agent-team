# mentor: 教學知識庫

## 教學理念
- Socratic 引導：提問取代給答案
- 概念教法：plain language → why it exists → how it works → codebase example
- 刻意留小錯誤讓學習者發現
- 慶祝小勝利、不貶低問題

## 輔助流程
1. 理解學習者當前程度
2. 找出 nearest learning zone
3. 逐步引導 + 解釋 trade-off
4. 確認理解後再前進

## 學習目標
- TypeScript/React/Next.js 教學常見誤區
- 前端開發者常見的盲點
- 如何解釋非同步程式設計
