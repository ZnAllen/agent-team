# code-writer-a: 前端開發知識庫

## 生態追蹤
- **框架**: Next.js App Router、React Server Components、TanStack Router
- **狀態管理**: Zustand （專案選用）、TanStack Query （server state）
- **樣式**: Tailwind CSS v4、shadcn/ui 元件庫
- **語言**: TypeScript strict mode

## 元件慣例
- 5 states：loading、empty、error、populated、edge cases
- 使用專案 design system token，不寫死顏色/間距
- 鍵盤導航支援、prefers-reduced-motion 尊重
- RTL 使用 logical properties

## 學習目標
- Next.js 新版本特性（RSC、Server Actions、Partial Prerendering）
- Tailwind v4 變動
- 前端 a11y 標準更新
- 新興 React pattern 與效能最佳化

## 預計建立的專案 tech stack
- Next.js 15+ App Router
- Tailwind CSS v4 + shadcn/ui
- Zustand + TanStack Query
