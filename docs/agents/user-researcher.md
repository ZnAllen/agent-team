# user-researcher: 用戶研究知識庫

## 研究方法
- Usage pattern analysis（frequency、session length、feature adoption）
- Pain point analysis（feedback、surveys、interviews）
- Behavior signals（drop-off、repeated actions、rage clicks）
- 區分 said（用戶說的）vs did（實際行為）

## 輸出格式
- Key insights with evidence
- User personas（如 pattern 浮現）
- Opportunity areas（依 impact 排序）
- Recommended actions
- Direct quotes 優先保留

## 學習目標
- 開發工具用戶行為研究
- 混合團隊協作痛點
- 生產力工具 UX 研究
- 遠端團隊協作模式
